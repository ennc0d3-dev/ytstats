# Extension Icons

Place your extension icons here:

- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

## Temporary Solution

For development/testing, you can create simple placeholder icons:

1. Use any image editor
2. Create PNG files with the above dimensions
3. Use a simple chart/stats icon or emoji

## Recommended

Create professional icons with:
- Bar chart or line graph symbol
- YouTube-style colors (red/white)
- Transparent background
- Clear and recognizable at small sizes

You can use tools like:
- Figma
- Canva
- Icon generators online
- Or hire a designer

For now, the extension will work without icons (Chrome will show a default placeholder).
